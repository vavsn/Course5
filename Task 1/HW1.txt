var a = 1, b = 1, c, d;
c = ++a; alert(c);           // 2
Выполнена префиксная операция инкремента

d = b++; alert(d);           // 1
Порядок вычислений: сначала значение b присваивается d, затем b увеличивается на 1

c = (2+ ++a); alert(c);      // 5
Порядок вычислений: сначала переменная а (2) увеличивается на 1 (получается 3), затем к значению а прибавляется 2.

d = (2+ b++); alert(d);      // 4
Порядок вычислений: к значению переменной b (2) прибавляется 2, т.к. запись b++ означает выполнение постфиксной операции. 
Увеличение значения b происходит уже после присваивания переменной d значения 4.

alert(a);                    // 3
Дважды производится префиксное увеличение переменной а на 1.

alert(b);                    // 3
Дважды производится постфиксное увеличение переменной b на 1.
